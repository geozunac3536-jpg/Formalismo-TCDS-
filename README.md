## Instrucciones rápidas
1. Reemplaza el ID de GA4 en `ga4_config.js` y en `index.html` por tu ID real.
2. Sube el ZIP a GitHub y habilita GitHub Pages (branch `main` o `docs/`).
3. Actualiza `sitemap.xml` si cambias la ruta de `index.html`.
4. En Zenodo, referencia esta página como *related identifier* (isSupplementTo).

## Métricas TCDS (Σ-metrics / K-Rate / LBCU)
- LI ≥ 0.9, R ≥ 0.95, RMSE_SL ≤ 0.1, reproducibilidad ≥ 95 %.
- K-Rate: Z ≥ 5 frente a base.
- LBCU local por compuerta: Q·Σ = φ.
